# Audio Similarity Analysis

## Project Overview
This project compares two audio recordings and calculates how similar they are using **MFCC features** and **frequency-domain analysis**.  

The project generates:
- **Time-domain waveform comparison**
- **Frequency-domain comparison**
- **MFCC heatmaps**
- **Similarity score** displayed on the graphs

## Requirements
- Python 3.x
- Libraries: `numpy`, `librosa`, `matplotlib`, `scipy`, `soundfile`

---

## Installation

Open your terminal or PowerShell and run:

```bash
# Install required libraries
pip install numpy librosa matplotlib scipy soundfile
````

---

## How to Run

1. Make sure your audio files are in the `audio/` folder
2. Run the Python script:

```bash
python run_project.py
```

3. The program will:

* Print the similarity score in the terminal
* Show the comparison plots
* Save the output image as `frequency_comparison.png`
